## 📌 Introduction

Customer segmentation is a critical business strategy that helps organizations understand different customer groups and tailor services accordingly.

This project develops a comprehensive analytics framework using clustering and ensemble learning models to identify customer profiles and predict future segment assignments.

## 🎯 Problem Statement

Organizations collect large volumes of customer-related data but often struggle to extract actionable insights.

This project aims to:

- Discover customer groups automatically.
- Detect anomalies and outliers.
- Build predictive models for customer classification.
- Analyze influential factors affecting segmentation.
